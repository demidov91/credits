Любой человек может создать тикет, в котором опишет желаемый кредит и какие у него есть документы.
Система принимает решение. Если решение положительное, то с этим тикетом и всеми заявленными
документами можно сразу идти в банк. Срок хранения тикета -- 10(?) дней.

Калькулятор кредитных выплат

Админка: доступ по user/password, 5 закладок.

Операционист (тот, кто выдаёт бабки): список подтверждённых кредитов, может ставить галочку "Бабки выданы"

Специалист по работе с клиентами: заполнение формы с учётом принятых у клиента документов (+ тикет,
если есть). Нажатие кнопочки ОК для отправки по дальнейшим этапам. Возможно, ей также будет приходить
мыло о том, что всё подтверждено и клиент может получить бабки.

Сотрудник службы безопасности: список документов для проверки с галочкой "одобрямс", список открытых
кредитов с сортировкой по "проблемности"

Сотрудник кредитного комитета: список кредитов, по которым требуется его веское слово. Может давать своё
веское слово. Предположительно, как только решение по кредиту становится ясно, голосование закрывается.

Начальник: просмотр всех кредитов, возможность сортировки и фильтрации; CRUD кредитных продуктов;
статы: Num/Dist кредитов по продуктам, по категориям юзеров, по статусам, Num/Dist каждого продукта по
юзерам и по статусам; (за период времени) количество решений скоринга "да" по проблемным кредитам,
количество у каждого члена кредитного комитета голосов "Да" по проблемным кредитам.

Категории юзеров и статусы кредитов -- Дима Демидов.
